Introduction

What is Python?

Python is a high-level, interpreted programming language renowned for its simplicity and versatility. Created by Guido van Rossum and first released in 1991, Python emphasizes readability and ease of use, making it an ideal language for both beginners and experienced developers. Its syntax closely resembles natural language, which helps reduce the complexity of coding and enhances the developer's productivity.

  • Interpreted Language: Python code is executed line-by-line, which simplifies debugging and allows for interactive coding.
  • Dynamic Typing: Variables in Python do not require an explicit type declaration, making the language more flexible and dynamic.

Using print in Python

The print() function in Python is a built-in function used to display information to the console or standard output. It is one of the most commonly used functions for debugging and communicating information in a Python program.

Basic Syntax:

print(object, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

Printing "Hello World!"

To print the classic "Hello World!" message in Python, use the print() function as follows:

print("Hello World!")

Basic Concepts

  1. Variable Declaration and Assignment: In Python, you declare a variable by simply assigning it a value. Python does not require explicit declaration of the variable type; it infers the type based on the assigned value.
  2. Variable Naming Rules
    • Names must start with a letter (a-z, A-Z) or an underscore (_)
    • The subsequent characters can be letters, numbers (0-9), or underscores

Example of Using Variables:

username = "Alice" age = 30 is_member = True
Data Type

Data types in Python define the kind of value a variable can hold and the operations that can be performed on it. Python has several built-in data types, which can be categorized into several classes, including numeric types, sequence types, mapping types, set types, and boolean types.

Numeric Types

  1. int

  2. float

  3. complex

Arithmetic Operators

Arithmetic operators in Python are used to perform basic mathematical operations on numerical values. These operators are straightforward and provide the fundamental functionality required for arithmetic calculations.

  1. Addition (+)

    Adds two numbers.

    result = a + b

  2. Subtraction (-)

    Subtracts one number from another.

    result = a - b

  3. Multiplication (*)

    Multiplies two numbers.

    result = a * b

  4. Division (/)

    Divides one number by another, resulting in a floating-point number.

    result = a / b

Summary:

Arithmetic operators in Python provide essential functionality for performing mathematical operations. Understanding these operators and their precedence helps you write accurate and efficient expressions in your code.
